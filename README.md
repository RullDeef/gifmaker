# GifMaker

## Example gif

![](out.gif)

## Usage

1. Rewrite `Update` and `Render` methods in `main.cpp`
2. Rebuild project with command `make`
3. Run `GifMaker.exe`
4. Wait for output

## Requirements

1. g++
2. [imagemagick](https://imagemagick.org/index.php)
